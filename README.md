# Terminal Pollution

Pollute your screen.

## Usage:

Redirect the output of a command to a new window with `tpo`:

```bash
$ tpo find /
```
### Requires:
	xfce4-terminal
	uuid
	bash

## Install

Clone the repo:

```bash
$ git clone https://github.com/seanmorris/terminal-pollution.git
$ cd terminal-pollution
```

Run the install script:


```bash
$ sudo make install
```
